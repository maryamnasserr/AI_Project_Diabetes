# Diabetes Data Analysis

## Overview

This repository contains an Exploratory Data Analysis (EDA) project performed on a diabetes dataset containing demographic, behavioral, and clinical information for 100,000 individuals. The analysis focuses on understanding the relationships between health indicators and diabetes while applying data preprocessing, statistical analysis, and data visualization techniques using Python.

The project was completed using Jupyter Notebook as part of a data analytics course.

---

## Dataset

The dataset contains information about approximately 100,000 individuals, including:

- Age
- Gender
- BMI
- HbA1c Level
- Blood Glucose Level
- Hypertension
- Heart Disease
- Smoking History
- Race
- Geographic Location
- Diabetes Status

---

## Objectives

The objectives of this project are to:

- Explore the structure of the dataset.
- Perform data cleaning and preprocessing.
- Analyze numerical and categorical variables.
- Generate descriptive statistics.
- Examine relationships between health indicators.
- Visualize trends using charts and correlation analysis.
- Identify factors associated with diabetes.

## Key Findings

Some important insights obtained from the analysis include:

- Blood glucose level and HbA1c level show the strongest positive relationship with diabetes.
- Higher BMI is associated with an increased likelihood of diabetes.
- Older individuals are more likely to develop diabetes than younger individuals.
- Hypertension and heart disease are positively associated with diabetes.
- Race shows minimal linear correlation with diabetes in this dataset.
- Most individuals in the dataset do not have hypertension or heart disease.
- Smoking history contains a significant number of unknown values.
